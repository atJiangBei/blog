JavaScript

具有函数优先的轻量级，解释型或即时编译型的编程语言。t基于原型编程、多范式的动态脚本语言，支持面向对象、命令式、声明式、函数式编程范式.

TypeScript

TypeScript 是一种基于 JavaScript 构建的强类型编程语言，可在任何规模下为您提供更好的工具。

Vue

基于标准 HTML、CSS 和 JavaScript 构建，并提供了一套声明式的、组件化的编程模型，帮助你高效地开发用户界面。

React

React 让你可以通过组件来构建用户界面。你可以创建像 Thumbnail、LikeButton 和 Video 这样的组件。然后将它们组合成整个应用程序。
